
# Add API using Maven

```xml
<repositories>
  <repository>
      <id>luckydrops-repo</id>
      <url>https://pingu-io.github.io/luckydrops-api/</url>
  </repository>
</repositories>
```

```xml
<dependencies>
  <dependency>
      <groupId>io.pingu</groupId>
      <artifactId>luckydrops-api</artifactId>
      <version>1.13</version>
  </dependency>
</dependencies>
```
